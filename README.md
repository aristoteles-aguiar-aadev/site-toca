# Toca Site — Projeto de Landing Page

Site institucional para a **Toca — Equipe de Desenvolvimento Infantil**, com identidade visual moderna inspirada em produtos digitais premium, mas usando elementos reais da marca para evitar aparência genérica ou “cara de IA”.

## Stack

- React
- TypeScript
- Vite
- CSS puro
- Lucide React para ícones

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse: `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```

## Estratégia visual

### Cores extraídas das referências anexadas

- Azul Toca: `#6F8FCC`
- Azul escuro: `#142544`
- Rosa claro: `#FFE1DC`
- Creme: `#FFFAF7`
- Lavanda: `#DED6E6`
- Verde suave: `#D9ECE5`

### Direção de design

- Visual limpo, premium e responsivo.
- Tipografia grande e direta no hero.
- Cards com cantos arredondados, blur e sombra suave.
- Uso de imagens reais da clínica.
- Microcopy mais humana, sem frases genéricas de IA.
- CTAs focados em WhatsApp e avaliação.

## Conteúdo usado

- Especialidades visíveis nas peças anexadas: Psicologia, Psicopedagogia, Psicomotricidade, Fonoaudiologia, Terapia Ocupacional, Fisioterapia Motora e Fisioterapia Visual.
- Bio pública do Instagram adiciona Musicoterapia.
- Contatos visíveis na peça: telefone, WhatsApp e Instagram.
- Convênios visíveis na peça: Unimed, Abmais Saúde, São Camilo, Rede Saúde, Cafaz e ISSEC.

## Observação para produção

Antes de publicar, validar com a clínica:

1. Endereço completo.
2. Horários de funcionamento.
3. Regras de convênio por especialidade.
4. Política de privacidade/LGPD.
5. Autorização formal para uso das imagens com crianças.
